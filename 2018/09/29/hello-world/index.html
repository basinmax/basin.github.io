Hello Hexo

2018-09-29

Hexo

Hexo的简洁、神秘让我跃跃欲试，在使用过程中遇到了很多问题，整理此文，一是方便其他猿人搭建自己的博客，二是给自己的学习Hexo之旅做个总结。

1.介绍

Hexo是基于NodeJs的静态博客框架，简单、轻量，其生成的静态网页可以托管在Github和Heroku上。

  • 超快速度
  • 支持MarkDown
  • 一键部署
  • 丰富的插件
    下面以我的博客为例：https://basinmax.github.io/basin.github.io/

2.环境搭建

  • 2.1 [node] 检测是否成功
    $ node -v
  • 2.2 安装hexo
     //Windows
 $ npm install hexo-cli -g
//Mac OS
 $ sudo npm install hexo-cli -g

  • 2.3上手

    创建博客目录 username.github.io @username改成自己的

    $ hexo init username.github.io
$ cd limedroid.github.io
$ npm install

    生成静态页面

    $ hexo clean
$ hexo g
$ hexo s

    然后打开浏览器，输入地址 localhost:4000 即可看到效果

    上面的【g】 即generate/编译 【s】即serve/启动
